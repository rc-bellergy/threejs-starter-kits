# ThreeJS Starter Kit - Backdrop
![](preview.jpg)

## Setup
``` bash
# Develop and run on node v17.7.1
# Run the `nvm` if need:
nvm use v17.7.1

# Install dependencies (only the first time)
npm install

# Run the local server at localhost:8080
npm run dev

# Build for production in the dist/ directory
npm run build
```
